🔍 植物分类对比

直根 vs 须根

直根

一根粗壮主根，侧根细小
例：凤仙花、桃树、萝卜

须根

无粗主根，大量细根丛生
例：水稻、小麦、葱

木本 vs 草本

木本植物

茎坚硬木质化，多年生
例：樟树、梧桐、杨树

草本植物

茎柔软无木质，生命周期短
例：狗尾草、三叶草、水稻

开花 vs 不开花

开花植物

依靠种子繁殖
例：桃树、向日葵、水稻

不开花植物

无花无种子，依靠孢子繁殖
例：苔藓、蕨类、藻类

👃 鼻咽喉

📯 气管

输送空气到肺部，有纤毛清除灰尘

🌳 支气管

分左右两支进入肺部

🫁 肺部

吸气过程：

外界空气 → 鼻 → 咽喉 → 气管 → 支气管 → 肺（氧气进入血液）

呼气过程：

体内CO₂ → 肺 → 支气管 → 气管 → 咽喉 → 鼻 → 外界

🌏 生命科学

探索生命的奥秘 — 遗传变异、进化与生物多样性

📋

遗传

亲代外形特征传递给后代

子女像父母、双胞胎相似
⚡
🎨

变异

同种生物个体存在差异

同种苹果大小颜色不同

遗传和变异是生物多样性的基础。遗传保留了物种的基本特征，变异则产生了新的特征，二者共同推动生物演化。

遗传 + 变异 = 生物演化

🪨 化石证据

化石是生物进化的直接证据。通过化石，我们可以了解古代生物的样子。
